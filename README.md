# 🎬 IMDb Movie Data Analysis & Dashboard

## 📌 Overview
This project presents an in-depth analysis of IMDb movie data using an interactive Tableau dashboard. It explores trends in movie ratings, genres, box office performance, and audience engagement over time.

## 🛠️ Tools Used
- Tableau  
- Data Cleaning & Processing  
- Data Visualization  
- Exploratory Data Analysis  

## 📂 Project Structure
- dashboard/ → Tableau workbook (.twbx)  
- reports/ → Exported PDF (optional)  
- screenshots/ → Dashboard preview images  

## 📊 Key Insights
- Drama and Crime genres dominate both in volume and audience engagement  
- Highly rated movies (8.0+) consistently receive higher vote counts  
- Significant growth in movie production after the 1990s  
- Box office revenue shows strong upward trends over decades  
- Top movies like *The Shawshank Redemption* and *The Dark Knight* have exceptionally high audience engagement  

## 📈 Dashboard Features
- IMDb rating and Meta score comparison by genre  
- Movie distribution by release year  
- Box office (gross) trends over time  
- Top 10 movies by number of votes  
- Certificate-wise movie distribution  
- Detailed movie-level insights (title, genre, rating, duration, votes)  

## 🚀 How to Use
1. Open the Tableau workbook file (.twbx)  
2. Interact with filters and visualizations  
3. Explore trends across genres, years, and ratings
